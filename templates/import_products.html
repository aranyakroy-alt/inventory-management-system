CSV Format Requirements

Required Fields

  • Name - Product name (text)
  • SKU - Unique product code (text)
  • Price - Product price (number, no currency symbol)
  • Quantity - Stock quantity (whole number, 0 or positive)

Optional Fields

  • Description - Product description (text)
  • Supplier - Supplier name (text, will create if doesn't exist)

Important Notes

  • Existing Products: Products with matching SKUs will be updated
  • New Products: Products with new SKUs will be created
  • Suppliers: New suppliers will be created automatically if they don't exist
  • Transactions: Stock changes will create transaction records
  • Validation: Negative prices and quantities are not allowed
